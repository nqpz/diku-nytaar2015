# Overblik

Dette dokument har overblikket over indkøb og struktur af nytårsaften 2015 på
DIKU.  Se [madplan.md](madplan.md) for plan for madlavning.

Dette kan også bruges som en guide til næste års nytårsaftener.


## Antal deltagere

Der kommer 29.  Vi laver mad til ~32.

Forventet pris pr. person: 100-150 kr.

Loft: 29 * 150 kr. = 4350 kr.


## Madallergier

+ kokos
+ skaldyr
+ æg
+ nødder
+ korn
+ majs
+ rodfrugter (bortset fra kartofler og rødbeder)
+ rå frugt og grønt
+ fisk
+ bacon
+ øl i maden
+ en af deltagerne kan virkelig ikke lide ost

## Andet indkøb

+ Fryseposer
+ Bagepapir
+ Husholdningsfilm
+ Servietter
+ Isterningeposer
+ Store foliebakker til kød
+ Pynt og sjov: Terpentiner, stjernekastere og masser af bordbomber (dog helst
  noget der ikke er umuligt at rydde op bagefter)!
+ Chips
+ 6 flasker bobler  


## Andre betænkeligheder

+ Varmekasser til steg og bagte kartofler: Vi låner kantinens.
+ Papirsdug: Vi låner kantinens.
+ Champagneglas: Vi låner kantinens.
+ Piskeris: Man kan ikke altid stole på kantinens, så Niels tager sine med.
+ Viskestykker: Vi skal have egne med jvf. kantine-udlåns-reglerne.  Niels tager
  en bunke med.
+ Virkende stegetermometer til stegen.  Der er et i kantinen.
